Elliptic Curves

Curves of the form y^2 = x^3 + px + q for some p and q

The equation y^2 = x^3 - N^2 x, which we got from the three successive squares problem, is an elliptic curve!
Elliptic curves are different from ellipses. (They were first named "elliptic" because similar equations appear when trying to find the perimeter of an ellipse.)
An elliptic curve is the set of solutions to the equation y^2 = x^3 + px + q, for some p and q. Depending on the exact values of p and q, the graph of an elliptic curve might look like one smooth curve, or two distinct parts.
(Technical note: For our purposes x, y, p, and q are all real numbers.)
In our case, we get a different curve for each value of N. When N=6, for example, we get the curve y^2 = x^3 - 36 x, which I've graphed here.
Here's the key idea: every triangle with area 6 corresponds to a point (x,y) on this curve.
Here's how to convert from triangles to points on this curve. First, let's start with this rational triangle with area 6.
From this triangle, we can compute r^2, s^2 and t^2.
During our algebra, we renamed s^2 to x.
And finally, we can use that x to plot a point on this elliptic curve.
As long as the triangle has area 6, we're guaranteed to get a point on this elliptic curve.
We can work backwards, too: if we find an x, we can compute r^2, s^2, t^2, which can be turned into a triangle with area N.
But more importantly: if we can find a point (x,y) where both x and y are rational, we can construct a rational triangle with area 6 - proving 6 is a congruent number!
Therefore, we've turned the problem of "is N a congruent number?" into "Does the equation y^2 = x^3 - N^2 x have any pairs of rational numbers (x,y) which solve the equation?"
That's great for us - mathematicians have years of experience finding solutions to equations!
Another reason elliptic curves are useful is they have a special property: you can generate new solutions (x,y) from old ones!
For us, this means once we find one rational triangle with area N, we can use the elliptic curve to generate more rational triangles with area N!
