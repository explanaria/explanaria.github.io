Tunnell's Theorem (1983)


If N is odd:

Does x^2+2y^2+8z^2=N have exactly twice as many integer solutions as x^2 + 4y^2 + 8z^2 = \frac{N}{2}?

If N is even:

Does x^2+2y^2+32z^2=N have exactly twice as many integer solutions as x^2 + 4y^2 + 32z^2 = \frac{N}{2}?

If so, N is a congruent number!

The equation y^2 = x^3 - N^2 x, which we got from the three successive squares problem, is an elliptic curve!
(Elliptic curves are different from ellipses. They were first named "elliptic" because similar equations appear when trying to find the perimeter of an ellipse.)
Previous Chapter
An elliptic curve is the set of solutions to the equation y^2 = x^3 + px + q, for some p and q. Depending on the exact values of p and q, the graph of an elliptic curve might look like one smooth curve, or two distinct parts.
(Technical note: For our purposes x, y, p, and q are all real numbers.)
In our case, we get a different curve for each value of N. When N=6, for example, we get the curve y^2 = x^3 - 36 x, which I've graphed here.
Here's the key idea: every triangle with area 6 corresponds to a point (x,y) on this curve.
Here's how to convert from triangles to points on this curve. First, let's start with this rational triangle with area 6.
From this triangle, we can compute r^2, s^2 and t^2.
During our algebra, we renamed s^2 to x.
And finally, we can use that x to plot a point on this elliptic curve.
As long as the triangle has area 6, we're guaranteed to get a point on this elliptic curve.
We can work backwards, too: if we find an x, we can compute r^2, s^2, t^2, which can be turned into a triangle with area N.
(Technical note: if x isn't a square number, we need to use a different equation than s^2 = x; we can always turn rational points into rational triangles and vice versa, but other equations don't have the same nice geometric interpretations.)
But more importantly: if we can find a point (x,y) where both x and y are rational, we can construct a rational triangle with area 6 - proving 6 is a congruent number!
Therefore, we've turned the problem of "is N a congruent number?" into "Does the equation y^2 = x^3 - N^2 x have any pairs of rational numbers (x,y) which solve the equation?"
That's great for us - mathematicians have years of experience finding solutions to equations!
Another reason elliptic curves are useful is they have a special property: you can generate new solutions (x,y) from old ones!
Let's say we found two rational points on this curve. Elliptic curves allow you to define a way to "add" two rational points on the curve, and get another, different rational point on the curve!
For us, this means once we find one rational triangle with area N, we can use the elliptic curve to generate more rational triangles with area N!
Here's the recipe for computing a new point P_1 + P_2. The definition of elliptic curve addition is a bit technical, but this recipe is defined the way it is so that adding points works similarly to adding numbers.
  • - First, draw a line through P_1 and P_2. Because of how elliptic curves are defined, This line will always* intersect the curve in a third point.
  • - Then, reflect that point vertically over the x-axis.
  • - The resulting point is called P_1 + P_2, and it's guaranteed to be a point on the elliptic curve.
      • (* Technically, vertical lines don't intersect the curve in three points, but mathematicians use a tool called the "projective plane" to take care of those cases.)
What if P_1 and P_2 are the same point? Just like normal addition, adding the same point to itself is possible! Instead of drawing a line between two points, when adding P_1 + P_1 you draw a tangent line to the curve. But everything still works out - we get a new point on the curve.
As it turns out, this "addition" obeys many of the same rules as normal addition. In particular, if you add two rational numbers points, you get another rational number point.
That's good for us - remember, each rational point on this elliptic curve corresponds to a rational-sided triangle with area N!
In other words, by using elliptic curve addition, once we find one rational triangle with area 6, we can construct a point on the elliptic curve, add it to itself, and generate new triangles with area 6!
We might need to zoom out to see P_1 + P_1, but the math still checks out - P_1 + P_1 is a rational point, which correspond to a valid rational triangle with area 6. And we can keep going - if we wanted to, now we could compute (P_1 + P_1) + P_1 and find another rational triangle with area 6.
By the way, modern encryption is built around adding points on elliptic curves!
Remember how I said your computer uses elliptic curves to encrypt its connection to the internet? In a nutshell, elliptic curve encryption weaponizes the fact that adding points on elliptic curves is really hard to undo. If someone adds a point P to itself repeatedly and hands you P + P + \ldots + P, it's almost impossible to find P, the point they started with. Elliptic curve encryption turns "undoing addition is hard" into "decrypting a message is hard"!
Finally, let me tell you about the solution to the congruent number problem! In 1983, a mathematician named J. B. Tunnell used deep properties of the elliptic curve y^2 = x^3 - N^2 x to solve the congruent number problem. His answer was a bit technical, so don't panic.
Here's Tunnell's answer: To figure out if N is a congruent number, count the number of integer solutions to these two equations and compare them.
Counting these solutions is simpler than it sounds - a computer can just try every single possible combination of x,y,z \leq \sqrt{N}. There's not a lot of possibilities!
What a complicated criterion. What do solutions of these equations have to do with whether or not N is a congruent number?
Here's why it works: Tunnell showed that if his criterion is invalid, it means the elliptic curve y^2 = x^3 - N^2 x has no rational solutions. And as we've seen, no rational solutions on the elliptic curve means no rational triangles with area N.
By the way, we started with a geometry question about triangles, and now we're answering it using number theory. Isn't that interesting? Mathematicians love elliptic curves because they connect to so many different areas of math.
For example, let's do the math and figure out whether N=5 is a congruent number.
Looking at the second equation, x^2+4y^2+32z^2=2.5, we can see there's no solutions: x^2, 4y^2 and 32z^2 are all integers, so there's no way they can add up to a non-integer like 2.5.
Similarly, for the first equation, it turns out x,y, and z can't be bigger than 2, or else x^2 + 2y^2 + 8z^2 \gt5. So we can check every single possibility by hand and find none of them solve the equation.
So there are 0 triplets of integers (x,y,z) which solve x^2+2y^2+8z^2=5,
and 0 triplets of integers (x,y,z) which solve x^2+2y^2+32z^2=\frac{5}{2}.
Since 0 = 2 \cdot0, Tunnell's criterion says 5 is a congruent number!
