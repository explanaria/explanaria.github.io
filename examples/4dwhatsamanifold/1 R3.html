Here's a point.
You're probably used to 2D coordinates. Two numbers, each denoting some distance away from the origin.
But you can also think about cartesian coordinates like this: they're all possible combinations of two independent numbers on the real line. When you see it like this, I think it's a bit more clear that they're independent of each other.
3D coordinates work the same way - only instead of two coordinates, we have 3.
Here, too, each set of 3D coordinates is really 3 sets of number lines, all of them independent from one another.
So coordinates are really just points on a number line.
